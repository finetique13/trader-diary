# Дневник трейдера

Вертикальный календарь для учёта торговых результатов.

## Запуск

```bash
npm start
```

Сайт будет доступен по адресу: **http://localhost:3000**

(Используется встроенный HTTP-сервер Python)

## Деплой

Папку можно разместить на любом хостинге статических сайтов:
- **GitHub Pages** — подключите репозиторий
- **Vercel** — `vercel` или через Git
- **Netlify** — перетащите папку в Netlify Drop
- Любой другой static hosting
